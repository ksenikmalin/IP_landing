Агентство по ремонту квартир

Ссылка на репозиторий проекта »

Задание

Темой проекта является веб-сервис для агентства по ремонту квартир.

В данной работе рассматривается удобная форма заказа услуги ремонта и их учета. Система создана для взаимодействия пользователей, менеджеров и администраторов.

Целью данного курсового проекта является разработка базы данных строительной организации, обеспечивающей автоматизацию функций пользователя в данной предметной области, что является более удобным и практичным в применении и создание административного интерфейса для неё.

Актуальностью данной темы появилась с развитием компьютерных технологий, благодаря чему возросла потребность хранения и обмена информацией. Её хранение играет важную роль в работе человека и предприятия. Ведь от достоверно правильной и доступной информации зависит успех его работы. В современном мире для хранения больших объёмов информации были разработаны базы данных (БД). Для более эффективного развития данной отрасли необходимо использовать автоматизированные информационные системы.

Для достижения данной цели были выявлены следующие задачи:

  1. Ознакомиться с литературой по данной теме и провести анализ предметной области «Агентство по ремонту квартир»;
  2. Анализ аналогичных сервисов;
  3. Проектирование и создание реляционной базы данных для веб-сервиса «Агентство по ремонту квартир»;
  4. Создание и настройка административной панели для управления базой данных с помощью фреймворка Django;
  5. Изучение формата Json и реализация RESTful API;
  6. Тестирование REST API с помощью инструмента Postman.

Используемые технологии


  1. Python – высокоуровневый язык программирования общего назначения, ориентированный на повышение производительности разработчика и читаемости кода. Синтаксис ядра Python минималистичен. В то же время стандартная библиотека включает большой объем полезных функций.
  2. Django – это программный каркас с богатыми возможностями, подходящий для разработки сложных сайтов и веб-приложений, написанный на языке программирования Python.
  3. SQLite — это программная библиотека, которая реализует автономный, безсерверный, транзакционный механизм базы данных SQL с нулевой конфигурацией.
  4. REST (сокр. от англ. Representational State Transfer — «передача состояния представления») – архитектурный стиль взаимодействия компонентов распределенного приложения в сети. REST представляет собой согласованный набор ограничений, учитываемых при проектировании распределенной системы.
  5. Django REST framework – удобный инструмент для работы с rest основанный на идеологии фреймворка Django.
  6. Django-import-export – это приложение и библиотека Django для импорта и экспорта данных с включенной административной интеграцией.
  7. Ckeditor – свободный WYSIWYG-редактор, который может быть использован на веб-страницах.
  8. Pillow – это свободно распространяемая библиотека для работы с изображениями на Python с открытым исходным кодом, которая добавляет коду поддержку открытия, изменения и сохранения изображений в различных расширениях.
  9. Json – текстовый формат обмена данными, основанный на JavaScript. Как и многие другие текстовые форматы, JSON легко читается людьми.
  10. Postman – популярная среди разработчиков программа для тестирования работы различных API. Она имеет графический интерфейс и поэтому намного удобнее консольной утилиты curl.
